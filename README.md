BallsUp!
========

BallsUp! is a vectorballs engine. So many balls!

![So Many Balls!](https://www.kranzky.com/img/portfolio/ballsup.png)

[Watch on YouTube](https://www.youtube.com/watch?v=rtaDkwrwLws)

```
$ brew bundle
$ make
$ ./bin/ballsup
```
